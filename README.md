# PDF質問アプリ

PDFファイルからテキストを抽出し、ユーザーの質問に回答するアプリです。フロントエンドはNext.js、バックエンドはFastAPIで構築されています。

## 機能

- **PDFファイルのアップロード**
- **質問に対する回答提供**

## プロジェクト構成

- **フロントエンド**: Next.js
- **バックエンド**: FastAPI

## セットアップ手順

### バックエンド

 `.env.sample` を `.env` にコピーし、OpenAI APIキーを設定します。

### フロントエンド

　`.env` にバックエンドのURLを設定します。
